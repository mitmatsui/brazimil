1.1 Variáveis — guardando dados econômicos

Em Python, uma variável é como uma célula nomeada no Excel. Você dá um nome e atribui um valor. A diferença é que pode guardar qualquer tipo de dado: número, texto, lista, dicionário.

Pythonvariáveis básicas
# Dados da Brazimil — variáveis numéricas
preco_venda    = 58.00    # Econômica 15L Balde BN
custo_fabrica  = 32.78    # custo de MP + embalagem
peso_kg        = 18.70    # peso bruto em kg
frete_por_ton  = 100.0    # R$/tonelada

# Variável de texto (string)
nome_produto   = "Econômica 15L Balde BN"
categoria      = "Tinta Base Água"

# Variável booleana (verdadeiro/falso)
em_producao    = True

# Exibir no Terminal
print(nome_produto)           # → Econômica 15L Balde BN
print(f"Preço: R$ {preco_venda}")  # → Preço: R$ 58.0
💡 Analogia com Excel: preco_venda = 58.00 é como digitar 58,00 na célula B2 e chamar essa célula de "preco_venda". Em vez de usar B2 nas fórmulas, você usa o nome que faz sentido.

1.4 Funções — blocos reutilizáveis

Uma função em Python é como uma fórmula personalizada no Excel (LAMBDA). Você define uma vez e usa em qualquer lugar do código.

Pythonfunção de classificação de MC
def classificar_mc(mc_pct):
    """Classifica o status da MC de um produto."""
    if mc_pct >= 35: return "Saudável"
    if mc_pct >= 20: return "Atenção"
    if mc_pct >= 0:  return "Crítico"
    return "Prejuízo"

# Usar a função com qualquer produto
print(classificar_mc(32.1))   # → Atenção
print(classificar_mc(42.5))   # → Saudável
print(classificar_mc(-3.4))   # → Prejuízo

Uma lista é uma coluna do Excel — uma sequência de valores na mesma variável. Em Python, listas usam colchetes [ ].

media = sum(precos) / len(precos)

4.1 A função calcMC — coração do dashboard

Esta é a função mais importante do sistema. Ela recebe os dados brutos de um produto e retorna todos os indicadores econômicos calculados com a tributação ZFM correta.

Pythonfunção completa de cálculo de MC — ZFM Bens Finais
def calc_mc_zfm(preco, custo, peso_kg, ce=0.55, comissao=0.05, frete_ton=100):
    """
    Calcula a Margem de Contribuição com tributação ZFM correta.

    Parâmetros:
        preco      : preço de venda em R$
        custo      : custo de fábrica (MP + embalagem) em R$
        peso_kg    : peso bruto em kg
        ce         : Crédito Estímulo (padrão 55% para Bens Finais)
        comissao   : % de comissão de vendas (padrão 5%)
        frete_ton  : custo de frete em R$/tonelada (padrão R$100)

    Retorna: dicionário com todos os indicadores calculados
    """
    # ── Custo variável ────────────────────────────────────
    frete  = (peso_kg / 1000) * frete_ton   # frete rateado pelo peso
    cv     = custo + frete               # custo variável total

    # ── Tributação ZFM ────────────────────────────────────
    icms_debito  = 0.12                    # ICMS débito saída (alíq. AM)
    icms_entrada = 0.07                    # crédito insumos nacionais
    icms_liq     = max(icms_debito * (1 - ce) - icms_entrada, 0)
    # Com CE 55%: max(0.12 * 0.45 - 0.07, 0) = max(0.054 - 0.07, 0) = max(-0.016, 0) = 0.054

    fmpes  = 0.06 * icms_debito * ce      # 6% × 12% × CE = 0.396%
    uea    = 0.015                          # Universidade Estadual AM
    fti    = 0.01                           # FTI insumo nacional
    pis    = 0.0065                         # PIS Lucro Presumido
    cofins = 0.03                           # COFINS Lucro Presumido

    carga_trib = icms_liq + fmpes + uea + fti + pis + cofins + comissao

    # ── Margem de Contribuição ────────────────────────────
    plv    = preco * (1 - carga_trib)       # Preço Líquido de Venda
    mc_r   = plv - cv                      # MC em R$
    mc_pct = (mc_r / plv * 100) if plv > 0 else 0

    # ── Preço mínimo (MC alvo 25%) ────────────────────────
    mc_alvo = 0.25
    preco_min = cv / ((1 - carga_trib) * (1 - mc_alvo))

    return {
        "frete":      round(frete, 2),
        "cv":         round(cv, 2),
        "carga_trib": round(carga_trib * 100, 2),  # em %
        "plv":        round(plv, 2),
        "mc_r":       round(mc_r, 2),
        "mc_pct":     round(mc_pct, 2),
        "preco_min":  round(preco_min, 2),
        "status":     classificar_mc(mc_pct),
    }

# Testar com a Econômica 15L BN
r = calc_mc_zfm(preco=58.00, custo=32.78, peso_kg=18.70)
print(f"Carga tributária: {r['carga_trib']}%")   # → 16.95%
print(f"PLV: R$ {r['plv']}")                   # → R$ 41.18
print(f"MC: R$ {r['mc_r']} ({r['mc_pct']}%)")  # → R$ 6.53 (15.8%)
print(f"Preço mínimo: R$ {r['preco_min']}")    # → R$ 64.70
print(f"Status: {r['status']}")               # → Crítico

4.2 Análise comparativa — CE correto vs modelo simplificado

Pythonquanto a MC estava subestimada no modelo antigo
def comparar_modelos(preco, custo, peso_kg, comissao=0.05):
    """Compara CE 55% correto vs modelo simplificado de 7%."""

    # Modelo correto (CE 55%)
    correto = calc_mc_zfm(preco, custo, peso_kg, ce=0.55, comissao=comissao)

    # Modelo simplificado (ICMS 7% simples, sem CE)
    frete       = (peso_kg / 1000) * 100
    cv          = custo + frete
    carga_simp  = 0.07 + 0.0065 + 0.03 + comissao  # 7%+PIS+COFINS+Com
    plv_simp    = preco * (1 - carga_simp)
    mc_simp     = (plv_simp - cv) / plv_simp * 100 if plv_simp > 0 else 0

    diferenca = correto["mc_pct"] - mc_simp

    print(f"{'MC modelo correto (CE 55%):':35} {correto['mc_pct']:+.2f}%")
    print(f"{'MC modelo simplificado (7%):':35} {mc_simp:+.2f}%")
    print(f"{'Diferença (MC subestimada):':35} {diferenca:+.2f} pp")
    print(f"{'Preço mínimo correto:':35} R$ {correto['preco_min']:.2f}")

print("=== Econômica 15L BN ===")
comparar_modelos(58.00, 32.78, 18.70)

5.1 O que é um CSV e como o Python o lê

CSV (Comma-Separated Values) é um arquivo de texto onde cada linha é um registro e os campos são separados por vírgula ou ponto-e-vírgula. É o formato universal de exportação do Omie e do Bradesco.

Pythonlendo um CSV de vendas do Omie
import csv
from datetime import datetime

# Abrir o CSV com encoding UTF-8 (padrão do Omie)
with open("vendas_omie.csv", encoding="utf-8-sig", errors="replace") as f:
    # DictReader lê cada linha como dicionário (coluna → valor)
    reader = csv.DictReader(f, delimiter=";")

    # Ver os nomes das colunas
    print(reader.fieldnames)
    # → ['Data Emissão', 'Nº NF', 'Cliente', 'Produto', 'Qtd', 'Valor Total']

    total_faturado = 0
    por_produto    = {}  # acumula faturamento por produto

    for linha in reader:
        # Acessar campos pelo nome da coluna
        produto = linha["Produto"]
        valor   = linha["Valor Total"]

        # Converter valor de texto para número
        # "1.234,56" → 1234.56
        valor_num = float(valor.replace(".", "").replace(",", "."))

        total_faturado += valor_num

        # Acumular por produto
        if produto not in por_produto:
            por_produto[produto] = 0
        por_produto[produto] += valor_num

print(f"Total faturado: R$ {total_faturado:,.2f}")

# Top 5 produtos por faturamento
top5 = sorted(por_produto.items(), key=lambda x: x[1], reverse=True)[:5]
print("Top 5 produtos:")
for prod, valor in top5:
    print(f"  {prod:35} R$ {valor:>10,.2f}")

                          else "aberto"

        except:
            continue

vencidos      = [b for b in boletos if b["status"] == "vencido"]
abertos       = [b for b in boletos if b["status"] in ["vencido", "aberto"]]
total_vencido = sum(b["valor"] for b in vencidos)
total_aberto  = sum(b["valor"] for b in abertos)

for ini, fim, label in [(1,30,"1–30 dias"),(31,60,"31–60 dias"),

6.1 Classe Balanço — estrutura de dados financeiros

Quando os dados ficam complexos, Python permite criar classes — estruturas que agrupam dados e funções relacionadas. É como criar um tipo de dado personalizado.

Pythonclasse Balanço para análise financeira da Brazimil
class Balanco:
    """Representa o balanço patrimonial de um período."""

    def __init__(self, periodo, ativo_circ, estoque, caixa,
                 passivo_circ, passivo_tot, ativo_tot, pl, lucro_liq, faturamento):
        self.periodo      = periodo
        self.ativo_circ   = ativo_circ
        self.estoque      = estoque
        self.caixa        = caixa
        self.passivo_circ = passivo_circ
        self.passivo_tot  = passivo_tot
        self.ativo_tot    = ativo_tot
        self.pl           = pl
        self.lucro_liq    = lucro_liq
        self.faturamento  = faturamento

    # ── Indicadores de Liquidez ───────────────────────────
    def liq_corrente(self):
        return self.ativo_circ / self.passivo_circ

    def liq_seca(self):
        return (self.ativo_circ - self.estoque) / self.passivo_circ

    def liq_imediata(self):
        return self.caixa / self.passivo_circ

    def capital_giro(self):
        return self.ativo_circ - self.passivo_circ

    # ── Indicadores de Rentabilidade ──────────────────────
    def margem_liquida(self):
        return self.lucro_liq / self.faturamento * 100

    def roe(self):
        "Return on Equity — retorno sobre o Patrimônio Líquido"
        return self.lucro_liq / self.pl * 100

    def roa(self):
        "Return on Assets — retorno sobre Ativos Totais"
        return self.lucro_liq / self.ativo_tot * 100

    # ── Indicadores de Endividamento ──────────────────────
    def grau_endividamento(self):
        return self.passivo_tot / self.ativo_tot * 100

    def autonomia(self):
        return self.pl / self.ativo_tot * 100

    def relatorio(self):
        """Imprime relatório completo formatado."""
        print(f"\n{'='*50}")
        print(f"RELATÓRIO FINANCEIRO — {self.periodo}")
        print(f"{'='*50}")
        print(f"LIQUIDEZ")
        print(f"  Corrente:  {self.liq_corrente():.2f}x  (ref: ≥1,50)")
        print(f"  Seca:      {self.liq_seca():.2f}x  (ref: ≥1,50)")
        print(f"  Imediata:  {self.liq_imediata():.2f}x  (ref: ≥0,20)")
        print(f"  Capital de Giro: R$ {self.capital_giro():,.0f}")
        print(f"RENTABILIDADE")
        print(f"  Margem Líquida: {self.margem_liquida():.1f}%  (ref: ≥10%)")
        print(f"  ROE:            {self.roe():.1f}%  (ref: ≥15%)")
        print(f"  ROA:            {self.roa():.1f}%  (ref: ≥8%)")
        print(f"ENDIVIDAMENTO")
        print(f"  Grau:     {self.grau_endividamento():.1f}%  (ref: ≤40%)")
        print(f"  Autonomia: {self.autonomia():.1f}%  (ref: ≥60%)")

7.1 O que é JSON e por que usamos

JSON (JavaScript Object Notation) é o formato que permite Python e JavaScript se comunicarem. O Python calcula os dados e escreve em JSON; o JavaScript lê esse JSON e exibe no navegador.

8.1 Gerando relatório em texto formatado

from datetime import datetime

def gerar_relatorio_executivo(produtos_calculados, periodo):
    """
    Gera um relatório executivo de precificação para apresentação.
    produtos_calculados: lista de dicionários com mc_pct, nome, etc.
    """
    total      = len(produtos_calculados)
    saudaveis  = [p for p in produtos_calculados if p["mc_pct"] >= 35]
    atencao    = [p for p in produtos_calculados if 20 <= p["mc_pct"] < 35]
    criticos   = [p for p in produtos_calculados if p["mc_pct"] < 20]
    mc_media   = sum(p["mc_pct"] for p in produtos_calculados) / total

    relatorio = f"""
╔══════════════════════════════════════════════════════════╗
║          RELATÓRIO EXECUTIVO DE PRECIFICAÇÃO             ║
║          Brazimil Indústria de Tintas — {periodo:14}  ║
╚══════════════════════════════════════════════════════════╝

RESUMO EXECUTIVO
  Total de SKUs analisados : {total}
  MC média do portfólio    : {mc_media:.1f}%
  SKUs saudáveis (≥35%)   : {len(saudaveis)} ({len(saudaveis)/total*100:.0f}% do portfólio)
  SKUs em atenção (20–35%): {len(atencao)} ({len(atencao)/total*100:.0f}% do portfólio)
  SKUs críticos (<20%)    : {len(criticos)} ({len(criticos)/total*100:.0f}% do portfólio)

DESTAQUES POSITIVOS — Top 3 maior MC
"""
    top3 = sorted(produtos_calculados, key=lambda p: p["mc_pct"], reverse=True)[:3]
    for i, p in enumerate(top3, 1):
        relatorio += f"  {i}. {p['nome']:30} MC: {p['mc_pct']:+.1f}%\n"

    if criticos:
        relatorio += f"\n⚠️  AÇÃO NECESSÁRIA — Produtos com MC crítica\n"
        for p in sorted(criticos, key=lambda x: x["mc_pct"])[:5]:
            relatorio += f"  → {p['nome']:30} MC: {p['mc_pct']:+.1f}%"
            relatorio += f"  Preço mín.: R$ {p.get('preco_min', 0):.2f}\n"

    relatorio += f"\nGerado em: {datetime.now().strftime('%d/%m/%Y %H:%M')}"
    return relatorio

def simular_desconto(nome, preco_base, custo, peso_kg,

    for desc in descontos:

        r = calc_mc_zfm(preco_desc, custo, peso_kg)
